HTML

  • Hypertext Markup Language (HTML)is used to create the structure of a webpage
    • It is made up of elements which work together to provide content and the basic framework for all pages on the web
  • The head element contains information about the webpage
    • Examples of tags in the head element include:
      • meta
      • title
      • script
  • The body element represents the visible content shown to the user
  • In Semantic HTML element, the name of the element describes its content and the role it plays on a webpage

CSS

  • Cascading Style Sheets (CSS) is a programming language used to define how webpage content is presented to users
    • CSS defines the colors, aesthetics, visual layout, and organization
  • There are three ways to style a webpage using CSS
    • Inline CSS
      • Style an element directly in the HTML file
    • Internal CSS style sheet
      • Embed an entire stylesheet directly in the HTML file by using a "style" element in the "head" element
    • External CSS style sheet
      • An external style sheet that is linked to the HTML file in the "head" element
      • PREFERRED METHOD!
  • CSS is organized in the following way:
    • Selector {property: value}
    • Ex:
      • div {color: blue}
    • CSS rules can be designated by using element names, class attributes, and ID attributes as selectors
    • The * selector declares that ALL elements on the page will have the designated properties
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • CSS Box Model

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • It is very important to git add -A (add to staging area) and git commit -m "message" often to help with version control
  • git pull origin main: pulls code from main branch to make sure local information matches
  • git push origin [feature branch]: pushes code from local environment to be reviewed and eventually merged with main branch
